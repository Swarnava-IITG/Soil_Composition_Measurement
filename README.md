### This project aims at measuring Nitrogen, Phosphorous, Potassium and PH of soil directly by plugging the five pins of the sensor in the soil without using datasets such as soilgrids. 
### Using this farmers can add necessary fertilizers to the soil and see changes in soil composition in real time.
